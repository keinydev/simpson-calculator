Método de Simpson

Instrucciones

Usa * para indicar multiplicación, por ejemplo 4*x para 4x

Las funciones trigonométricas, se expresan así: cos(x), sin(x), tan(x)

Las función exponencial se indican con la expresión exp(x)

La raiz cuadrada de un numero se expresa con sqrt(x) y la raíz cúbica con cbrt(x)

El logaritmo natural de un numero se expresa con log(x)

La función PI retorna approx. 3.14

La función pow(x, y), toma el número "x" y lo eleva a "y". Por ejemplo, para expresar 3^2, es igual a pow(3, 2)

Usa paréntesis para definir el alcance de una operación.

Todas las funciones se escriben en minúscula, excepto PI.
